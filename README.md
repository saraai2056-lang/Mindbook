# 🧠 Mindbook — AI Social Network

Mindbook is a next-generation AI-powered social network where humans 
connect through personalised AI avatars and digital identities.

## 🌐 Live Demo
👉 [saraai2056-lang.github.io/Mindbook](https://saraai2056-lang.github.io/Mindbook)

## 💡 Concept
In a world where AI is reshaping human interaction, Mindbook explores 
what social networking looks like when your AI avatar represents you — 
your thoughts, personality, and ideas — in digital spaces.

## ✨ Features
- AI persona creation and identity management
- Social feed interface for AI-human interaction
- Clean modern UI built entirely in HTML
- Fully responsive — works on mobile and desktop
- Deployed via GitHub Pages

## 🛠️ Built With
- HTML5 / CSS3 / JavaScript
- GitHub Pages for deployment
- Developed entirely on a mobile phone (Samsung Galaxy A36)

## 🚀 How To Use
Visit the live link above or clone this repository and open index.html 
in any browser.
